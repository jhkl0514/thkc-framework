상품상세 페이지

• 상품상세 페이지는 고객이 상품에 대한 모든 정보를 확인하고 구매 결정을 내리는 중요한 공간이다.

• 유저의 시선을 사로잡고 구매 욕구를 자극할 수 있도록 효과적인 구조를 갖추는 것이 중요하다.

구조

a. 대대여 기준 : 설치 건
  • • 상품이미지:

    주 이미지 (510 X 510), 썸네일 이미지 (70 X 70)

  • • 상품정보:

    상품명, 금액, 상품상세, 지역선택, 배송희망일, 설치옵션, 청구 정보로 구성

b. 대대여 기준 : 택배 건
  • • 상품이미지:

    주 이미지 (510 X 510), 썸네일 이미지 (70 X 70)

  • • 상품정보:

    상품명, 금액, 상품상세, 청구 정보로 구성

상품상세 페이지

• 상품상세 페이지는 고객이 상품에 대한 모든 정보를 확인하고 구매 결정을 내리는 중요한 공간이다.

• 유저의 시선을 사로잡고 구매 욕구를 자극할 수 있도록 효과적인 구조를 갖추는 것이 중요하다.

구조

a. 대대여 기준 : 설치 건
  • • 상품이미지:

    PC에서 등록된 이미지 동일하게 사용, 디바이스 너비에 맞게 썸네일 이미지 확대

  • • 상품정보:

    상품명, 급여가, 배송희망일, 이용료, 본인부담금, 상품상세, 상품 이미지로 구성됨

b. 대대여 기준 : 택배 건
  • • 상품이미지:

    PC에서 등록된 이미지 동일하게 사용, 디바이스 너비에 맞게 썸네일 이미지 확대

  • • 상품정보:

    상품명, 급여가, 이용료, 본인부담금, 배송안내, 상품상세, 상품 이미지로 구성됨

사이즈 및 간격

• 기본 : padding : 24px, 20px , gap : 4px

• 아코디언 경우 : padding : 8px, 20px , gap : 4px

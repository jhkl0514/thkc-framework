상품리스트 페이지

상품 리스트 화면은 사용자가 다양한 상품을 탐색하고 선택하여 구매할 수 있도록 돕는 페이지이다.
이 화면은 사용자에게 제품 정보를 제공하고, 필터링 및 정렬 기능을 통해 원하는 상품을 쉽게 찾을 수 있게 해야 한다.

주요 구성 요소

  • • 상품 필터 및 검색:

    카테고리, 검색 기능을 제공한다.

  • • 상품 정렬 및 개수:

    상품 정렬 (추천수/ 높은가격순/ 낮은가격순/ (대여많은순)), 상품 개수 (총 개수)를 제공하다.

  • • 상품 리스트:

    상품이미지, 상품명, 상품관련 태그, 가격, 상태값 태그를 보여준다.

  • • 페이지 네비게이션:

    여러 페이지에 걸쳐 상품을 나누어 표시할 때 페이지 번호를 제공하여 사용자가 쉽게 탐색할 수 있게 합니다.

구조

a. 대대여 기준 : 상품 필터 및 검색
  • 1상품 필터 : Tap (탭) 컴포넌트 바로가기
  • 2검색 : Search (검색) 패턴 바로가기
b. 대대여 기준 : 상품 정렬 및 개수
  • 1상품 개수
  • 2상품 정렬 : Select (셀렉트) 컴포넌트 바로가기
C. 페이지네비게이션

• 노출 : 상품 20개 초과 (21개부터) 노출 함.

  • 1Pagination (페이지네이션) : 컴포넌트 바로가기
D. 대대여 기준 : 상품 리스트

• 배치 : 상품 4x 3 (12개)

• 간격 : 위 아래 gap 40px, 좌 우 gap 20px

  • 1Card (카드) : 컴포넌트 바로가기

상품리스트 페이지

상품 리스트 화면은 사용자가 다양한 상품을 탐색하고 선택하여 구매할 수 있도록 돕는 페이지이다.
이 화면은 사용자에게 제품 정보를 제공하고, 필터링 및 정렬 기능을 통해 원하는 상품을 쉽게 찾을 수 있게 해야 한다.

구조

a. 대대여 기준 : 설치 건
  • • 상품이미지:

    PC에서 등록된 이미지 동일하게 사용, 디바이스 너비에 맞게 썸네일 이미지 확대

  • • 상품리스트 :

    상품 배치 2 x 2 (무한 스크롤) : 사용자가 화면을 아래로 스크롤하면 새로운 상품들이 계속해서 나타나는 방식

  • • 간격 :

    위 아래 gap 32px, 좌 우 gap 16px

b. 대대여 기준 : 택배 건

사이즈 및 간격

• 기본 : padding : 4px, 20px, 20px,20px, gap : 16px, 32px

• 리스트 : gap : 8px

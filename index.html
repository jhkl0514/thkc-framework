이로움케어 프레임워크

장기요양기관 통합관리 플랫폼 '이로움Care'의 UI/UX를 위한
가이드라인으로, 일관성 있는 디자인과 효율적인 구축을 지원합니다.

주요 가이드

  • 파운데이션 Foundation
  • 컴포넌트 Components
  • 패턴 Pattern

가이드 검색

  • 색상 Color

    이로움케어 서비스를 상징하며 스타일을 표현하고 의미를 전달한다.

    파운데이션 스타일
  • 서체 Typography

    서체 사용의 원칙을 지켜, 콘텐츠를 일관적으로 보여지도록 한다.

    파운데이션 스타일
  • 형태 Shape

    형태의 고유 특징을 나타내어 UI를 개성있게 유지한다.

    파운데이션 스타일
  • 배치 Layout

    간격과 구분에 대한 기준을 제시해서 특유의 일관성을 부여한다.

    파운데이션 스타일
  • 아이콘 System Icon

    간단한 동작, 기능을 나타내며 효율적으로 정보를 전달한다.

    파운데이션 스타일
  • 브레드크럼 Breadcrumb

    탐색 계층 구조를 표시하여 사용자가 현재 위치를 파악하게 해준다.

    컴포넌트 탐색
  • 헤더/네비 GNB

    탐색 계층 구조를 표시하여 사용자가 현재 위치를 파악하게 해준다.

    컴포넌트 탐색
  • 사이드 메뉴 Side Navigation

    서브 화면 내에서의 이동을 위해 사용하는 메뉴이다.

    컴포넌트 탐색
  • 페이지네이션 Pagination

    많은 콘텐츠를 탐색하기 쉽게 여러 화면에 나누고, 탐색하는 데 사용한다.

    컴포넌트 탐색
  • 아코디언 Accordion

    중요한 세부 사항을 먼저 표시하고 확장되면 더 많은 세부 정보를 표시해 점진적으로 정보를 공개

    컴포넌트 레이아웃
  • 아바타 Avatar

    사용자에 관한 정보를 제공하는 역할로 사용된다.

    컴포넌트 레이아웃
  • 배지 Badge

    컴포넌트에 대한 빠른 인지와 탐색을 돕기 위한, 작은 데이터이다.

    컴포넌트 레이아웃
  • 달력 Calendar

    날짜와 관련된 정보와 기능을 제공하는 데 사용한다.

    컴포넌트 레이아웃
  • 카드 Card

    정보 제공을 목적으로 한 컨테이너이다.

    컴포넌트 레이아웃
  • 캐러셀 Carousel

    하나의 콘텐츠 영역 내에 여러 개의 서로 다른 콘텐츠를 표시한다.

    컴포넌트 레이아웃
  • 레이블 Label

    관련 구성요소에 대한 설명 텍스트를 제공하여 사용자 이해를 돕는다.

    컴포넌트 레이아웃
  • 모달 Modal

    별도의 화면을 통해 중요 정보를 전달하거나 작업을 완료하도록 돕는다.

    컴포넌트 레이아웃
  • 상태 칩 Status Chips

    주문완료, 출고대기, 주문완료와 같이 상태가 변하는 값에 사용한다.

    컴포넌트 레이아웃
  • 탭 Tabs

    동일 계층의 콘텐츠를 그룹화해 화면 이동 없이 빠른 탐색을 돕는다.

    컴포넌트 레이아웃
  • 표 Table

    데이터를 열과 행으로 구성된 그리드에 표시해, 효율적으로 정리한다.

    컴포넌트 레이아웃
  • 버튼 Button

    선택을 통해 상호작용을 유도하거나 작업을 수행할 수 있다.

    컴포넌트 액션
  • 링크 Link

    다른 서비스나 다른화면, 다른 섹션 등으로 이동하는데 사용된다.

    컴포넌트 액션
  • 체크박스 Checkbox

    사용자가 여러 개의 옵션 중 한 개 이상의 값을 선택할 수 있도록 한다.

    컴포넌트 선택
  • 칩 Chips

    사용자로 하여금 정보, 선택, 콘텐츠의 필터링과 특정 액션을 유도한다.

    컴포넌트 선택
  • 라디오 버튼 Radio button

    사용자가 여러 옵션 중 한 개의 값을 선택할 수 있도록 한다.

    컴포넌트 선택
  • 세그먼트 컨트롤 Segmented Control

    라디오 버튼과 유사한 기능을 제공하며, 일반적으로 상호 배타적인 옵션을 표시한다.

    컴포넌트 선택
  • 셀렉트 Select

    사용자가 여러 옵션 중 하나를 선택할 수 있게 목록을 팝업으로 제공한다.

    컴포넌트 선택
  • 스위치/토글 Switch/Toggle

    사용자가 두 가지 상호 배타적인 옵션 중 하나를 선택하도록 돕는다.

    컴포넌트 선택
  • 텍스트 영역 Textarea

    사용자가 두 줄 이상의 텍스트를 입력하는 경우에 사용하는 요소이다.

    컴포넌트 입력
  • 텍스트 입력 필드 Text input

    글자, 숫자, 기호 등이 조합된 한 줄의 짧은 텍스트를 입력하는 경우에 사용한다.

    컴포넌트 입력
  • 파일 업로드 File upload

    하나 이상의 디바이스의 로컬 파일을 선택하고 첨부하는 데 사용한다.

    컴포넌트 입력
  • 로딩 Loading

    화면이나 데이터에 접근하기 위해 대기해야 함을 사용자에게 안내한다.

    컴포넌트 피드백
  • 단계표시기 Step indicator

    서비스 이용을 위해 거쳐야하는 진행 단계를 시각화하여 표현한다.

    컴포넌트 피드백
  • 토스트 Toast

    간단한 알림 메시지로 즉각적이고 일시적인 피드백을 제공한다.

    컴포넌트 알림
  • 툴팁 Tooltip

    특정 영역에 대한 도움말이나 부가 정보를 제공한다.

    컴포넌트 알림
  • 검색 Search

    사용자가 큰 데이터 집합에서 원하는 정보를 찾을 수 있도록 도와준다.

    패턴 검색
  • 알림 Message

    사용자에게 정보를 전달하거나 상호작용을 유도하기 위해 사용한다.

    패턴 알림
  • 오류 Error

    사용자가 요청한 작업을 완료하지 못하는 문제가 발생했을 때 제공된다.

    패턴 오류
  • 입력폼 Forms

    데이터를 입력하여 서버로 전송하는 데 사용하는 콘텐츠 섹션이다.

    패턴 입력폼
  • 상품리스트 Item List

    다양한 상품을 탐색하고 선택하여 구매할 수 있도록 돕는다.

    패턴 주문
  • 상품상세 Item detail

    사용자의 구매 결정을 위해, 상품에 대한 자세한 정보를 제공한다.

    패턴 주문
  • 주문정보확인 Order

    사용자의 주문 내용을 상세히 검토하고 최종 확인을 위한 페이지이다.

    패턴 주문
  • 테이블_기본형  

    데이터나 정보를 구조적으로 정리해 놓은 표이다.

    패턴 테이블
  • 테이블_데이터 입력형  

    사용자가 데이터를 입력하거나 수정할 수 있도록 구성된 테이블이다.

    패턴 테이블
  • 테이블_조합형  

    기본형과 데이터 입력형이 합쳐진 형태의 표이다.

    패턴 테이블
  • 카드_모바일 Card

    모바일 웹에서의 콘텐츠를 시각적으로 분리된 카드 형태로 구성한다.

    패턴 카드
  • 오류 Error

    사용자가 요청한 작업을 완료하지 못하는 문제가 발생했을 때 제공된다.

    패턴 오류
  • 빈화면 Blank screen

    콘텐츠가 아직 들어가지 않은 경우, 오류라고 느끼지 않도록 안내한다.

    페이지 빈화면

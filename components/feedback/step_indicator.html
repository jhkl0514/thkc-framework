  • Components
  • Feedback (피드백)
  • Step indicator (단계 표시기)
Design Code

Step indicator

• 서비스 이용을 위해 사용자가 거쳐야하는 일련의 단계를 시각화하여 표현한 것으로 진행 상태에 대한 피드백을 사용자에게 전달한다.

구조

  • 1단계 문자 :

    단계를 표시하는 문자

  • 2현재 단계 식별자 :

    텍진행 중인 단계를 다른 단계와 구분하는 식별자

  • 3완료된 단계 식별자 :

    완료된 단계를 다른 단계와 구분하는 식별자

  • 4연결선 :

    단계와 단계 사이를 연결하는 선으로 프로세스의 선형성을 보여줌

상태

  • 1Completion :

    완료

  • 2Currently in progress :

    진행중

  • 3Before execution :

    진행전

사이즈 및 간격

가이드

• 단계는 좌에서 우 또는 상에서 하의 순서로 논리적으로 전개될 수 있도록 구성한다.

• 단계의 수는 최소 3개에서 최대 7개로 제공한다.

• 현재 단계를 명확하게 인지할 수 있도록 표현한다.

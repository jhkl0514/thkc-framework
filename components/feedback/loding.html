Design Code

Loading

• 페이지나 요소의 다양한 처리 상태를 시각적으로 표시한 것으로 화면 전체나 일부 요소에 접근하기 위해 일정 시간 동안 대기해야 함을 사용자에게 안내한다.

구조 : 라인 로딩 애니메이션

  • 1점선:

    점선이 왼쪽에서 오른쪽으로 움직임

  • 2파동:

    파동이 왼쪽에서 오른쪽으로 움직이면서 원이 커짐

사이즈 및 간격

가이드

• 브라우저 전체 화면 : 상/하/좌/우 정 가운데 배치

• 로딩 화면 효과 : 반투명 디머 (#A1A1A1, 투명도 40%)

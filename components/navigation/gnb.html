Design Code

GNB

• 사용자가 웹사이트에 접속하자마자 마주하게 되는 화면의 최상단 영역으로 서비스의 브랜드 이미지를 전달하는 핵심 영역이다.

• 모든 화면에 일관성 있게 배치되며 통합검색, 메인 메뉴 등 서비스 정보를 탐색하고 이동할 수 있는 핵심 탐색 수단을 제공한다.

구조

  • 1헤더 영역

    주요 메뉴, 서비스 식별정보, 사용자 정보를 제공하는 영역

  • 2전체 메뉴 버튼

    버튼을 클릭하면 ‘전체 메뉴'가 나타나거나 숨길수 있어 공간활용이 용이한 버튼

  • 3브랜드 로고

    브랜드 로고를 배치하여 서비스 채널의 정체성을 전달함

  • 4사용자 정보(아바타)

    사용자 정보 및 로그아웃 링크를 제공함

사이즈 및 간격

사용성 가이드

  • • 화면을 스크롤 했을 때 헤더를 뷰포트 상단에 고정한다.

  • 사용자가 긴 화면을 스크롤 하면서도 헤더에서 제공하는 탐색 및 유틸리티에 계속 접근 가능하도록 헤더를 고정하여 제공한다.

  • • 기본 디바이스 사이즈는 width 100%로 제공한다.
  • • 화면의 너비가 콘텐츠 사이즈보다 작을 경우 햄버거 버튼 모양이 바뀌면서 사이드 메뉴가 사라진다.

  • 작은 화면에서는 사용자가 본문 콘텐츠에 집중할 수 있도록 사이드 메뉴를 숨기고 햄버거 메뉴 버튼를 이용하여 메뉴를 플로팅형태로 보여준다.

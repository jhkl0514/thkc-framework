Design Code

Avatar

• 아바타는 사용자를 설명하는데 사용되는 구성 요소로, 사용자에 관한 정보를 제공하는 역할로 사용된다.

구조

  • 1컨데이너 :

    아이콘을 감싸고 있는 영역

  • 2아이콘 :

    사용자를 대표하는 아이콘 형태

사이즈

• SM, MD, LG, XL 4가지 사이즈를 지원한다.

사용법

  • • GNB 영역에서 사용자 정보 및 로그아웃 링크를 제공한다.

  • PC는 GNB영역에 노출하며 모바일에서는 사이드 메뉴에서 아바타를 노출한다.

  • • 테이블에서 사용자 구분값이 필요할 때 사용할 수 있다.

  • 사용자 구분값에 이름과 함께 아바타 아이콘을 함께 노출 할 수 있다.

Design

Link

• 링크는 다른 서비스/애플리케이션, 한 서비스 내의 다른 화면, 한 화면 내의 다른 섹션 등으로 이동하는 데 사용되는 탐색 요소이다.

구조

  • 1레이블 :

    링크와 연결된 대상을 안내하는 텍스트

  • 2아이콘(선택) :

    현재 서비스의 맥락을 벗어나 다른 웹사이트로 이동하거나 부가적인 도움말 제공 화면으로 이동하는 경우에 사용됨

상태

  • 1Default :

    기본

  • 2Hover:Active :

    호버/눌림/포커스

  • 3Disabled :

    비활성화

가이드

• 링크가 인라인으로 사용될 때 링크 텍스트를 주변 텍스트와 동일한 크기로 제공한다.

• 링크 레이블은 연결된 목적지 정보를 예측할 수 있는 내용으로 제공한다.

• 외부 서비스로 이동하는 링크를 분명하게 구분하고 일관된 방식으로 표현한다.

Breadcrumb (브레드크럼)

  • Components
  • Navigation (탐색)
  • Breadcrumb (브레드크럼)
Design Code

Breadcrumb

• 브레드크럼은 탐색 계층 구조를 표시하여 사용자가 현재 위치를 파악하게 해준다.

구조

  • 1경로 - 첫 번째 목록

    브레드크럼 목록의 첫 번째 항목을 보여줌

  • 2경로 - 상위 구조

    브레드크럼 목록의 첫 번째 항목을 보여줌

  • 3구분자

    브레드크럼 항목을 구분하기 위한 구분 기호. 화면 간의 시각적 계층 구조를 나타냄

가이드

• 브레드크럼 목록의 가장 마지막 항목은 다음 경로를 나타내는 [ > ] 아이콘을 표시하지 않는다.

• 표시 기준 : 현재 페이지를 기점으로 뒤로 3개까지 표시 (사용자가 5 Depth에 위치한 경우, 다음과 같이 3Depth, 4Depth, 5Depth만 BreadCrumb에 표시됩니다.)

• 예시 : 현재 페이지 Depth: 5, BreadCrumb 표시: 3Depth > 4Depth > 5Depth

• 모바일 : 브레드크럼 표시하지 않음

상태 (*TDB)

• 링크의 상태 값은 Default, Hover:Active:Press, Disabled 상태 값을 표현한다.

• 브레드크럼 목록의 가장 마지막 항목은 다음 경로를 나타내는 [ > ] 아이콘을 표시하지 않으며, 링크의 상태 값도 표현하지 않는다.

• 링크 설정 필요 : BreadCrumb에 표시되는 각 경로는 클릭 가능한 링크로 설정되어야 한다. (추후, Jira 티켓을 생성하여 링크 설정 작업을 진행할 예정)

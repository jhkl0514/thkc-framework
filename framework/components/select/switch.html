Design Code

Switch / Toggle

• 사용자가 두 가지 상호 배타적인 옵션 중 하나를 선택할 수 있도록 제공한다.

• 일반적으로 "온/오프", "켜짐/꺼짐", "활성/비활성"과 같은 상태를 나타내는 데 사용한다.

구조

  • 1트렉 :

    사용자에게 현재 선택된 상태를 명확하게 보여주는 요소로 직사각형 또는 타원형, 색상이나 패턴을 사용하여 상태를 구분함

  • 2핸들 :

    사용자가 직관적으로 상태를 변경할 수 있도록 도와주는 요소로 일반적으로 원형 또는 타원형이며, 트랙 위에 배치함

상태

  • 1Selecte/Active :

    선택 / 활성화

  • 2Unselecte/ Inactive :

    미 선택 / 비 활성화

가이드

• 선택 시 핸들이 트랙의 반대편으로 이동합니다.

• 상태 전환을 식별할 수 있는 색상 변화나 대비를 트랙에 적용합니다.
